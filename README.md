# `nomad-tool`

> A tool for quickly spinning up Nomad for development environments.

## Install

```bash
npm i -g nomad-tool
```

## Requirements

- Nomad
- Docker

## Usage

```bash
nomad-tool start
```

## Config

To see what the underlying configuration Nomad is using, see `.nomad.hcl`

```bash
cat .nomad.hcl
```
